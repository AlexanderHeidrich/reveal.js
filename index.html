Installation

Basic installation

Download the the latest release-ZIP from the GitHub page

Unpack and open the index.html in your browser

Full installation 

                    
                    git clone https://github.com/hakimel/reveal.js.git

                    cd reveal.js && npm install

                    npm start

                    // open http://localhost:8000

PDF-Export

Just open the presentation with ?print-pdf at the end of the URL.

Make sure to disable fragment-slide-separation, else you have one page per fragment/code separation. 

Reveal.configure({ pdfSeparateFragments: false });
