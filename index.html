Installation

Basic installation

Download the the latest release-ZIP from the GitHub page

Unpack and open the index.html in your browser

Full installation 

                    
                    git clone https://github.com/hakimel/reveal.js.git

                    cd reveal.js && npm install

                    npm start

                    // open http://localhost:8000

PDF-Export

PDF-Export

Just open the presentation with ?print-pdf at the end of the URL.
Make sure to disable fragment-slide-separation, else you have one page per fragment/code separation.
Reveal.configure({ pdfSeparateFragments: false });

Speakerview

Speakerview

Just press "S"

Speakernotes

Speakernotes

Add an aside-element with the notes-class to a slide.
This will then only appear in the speaker view.
This only works with the full installation and a running node-server.
Remember to tell them to disable notes on print view.

Speakernotes

You can also disable the notes on print page, or move them to an additional page: 

                        Reveal.initialize({
                            showNotes: true | false | "separate-page"
                        });

It also has an API

Use it to listen to events or move to slides, e.g. after finishing a video.
